# password-validator

# Live
https://arafal21.github.io/password-validator/

# Features
• You can hide and show your password
• You can see the controls that show you what is missing to make your password strong
• You will see the corresponding text under the program, whether your password is bad, medium or good.

# Technologies Used
• HTML
• SCSS / CSS
• JavaScript
